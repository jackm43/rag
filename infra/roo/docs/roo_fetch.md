## roo fetch

Invoke a registered application method as Connect JSON

### Synopsis

Invoke <app>.<Service>.<Method> as a Connect JSON request.

The request message is sent as Connect JSON; -d defaults to {} and accepts
inline JSON, @file, or @- for stdin. Use --stream for server-streaming RPCs;
streaming chat prints token text to stdout, --stream-json prints one JSON
object per chunk. Run "roo fetch <app>" to list the callable methods of an
application.

```
roo fetch <app>[.<Service>.<Method>] [flags]
```

### Options

```
      --as string     invoke while impersonating a service application
  -d, --data string   request body: inline JSON, @file, or @- for stdin (default "{}")
  -h, --help          help for fetch
      --stream        treat the method as a server-streaming RPC
      --stream-json   stream one JSON object per chunk
```
