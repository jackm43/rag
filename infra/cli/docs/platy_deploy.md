## platy deploy

Deploy workers from applications.yaml with 1Password secrets

### Synopsis

Deploys the workers declared in infra/applications/applications.yaml (all when no app is named).
Application secrets declared in the manifest are resolved through 1Password and
pushed to each worker; terraform client metadata is synced into the gateway config.
Applications whose inputs are unchanged since the last deploy are skipped.

```
platy deploy [app...] [flags]
```

### Options

```
      --force   deploy even when no inputs changed since the last deploy
  -h, --help    help for deploy
```
